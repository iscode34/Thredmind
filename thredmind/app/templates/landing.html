Frequently Asked Questions

How does the AI model rotation work?
ThredMind tries providers in order: DeepSeek → Google Gemini → NVIDIA → OpenRouter. If one fails, it automatically falls back to the next. You never notice an outage.
Where is my data stored?
All your data is stored in a PostgreSQL database you control. Embeddings run locally on your machine via ONNX Runtime — no API calls needed for search. AI generation uses external providers, but your document content is only sent during active generation.
What file formats are supported?
PDF, DOCX, PPTX, TXT, Markdown, and URL imports. More formats coming soon.
How does the knowledge graph work?
Documents are linked by shared entities, keywords, categories, and semantic similarity (via embeddings). The graph auto-updates whenever you upload new content, revealing connections between documents you might never have noticed.
